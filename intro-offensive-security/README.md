🧪 Intro to Offensive Security

🎯 Objetivo

Compreender os fundamentos da segurança ofensiva e o fluxo básico de um ataque em ambiente controlado.

🧠 Conceitos aprendidos

Reconhecimento de alvo
Noções de exploração
Fluxo básico de pentest
Importância da enumeração
🔍 Ferramentas introdutórias

Nmap (conceito básico)
frirefox (alterar parâmetro)

💡 Aprendizado

Este módulo introduziu a mentalidade de segurança ofensiva, mostrando que ataques são processos estruturados e não ações isoladas.
