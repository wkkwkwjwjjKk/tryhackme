🏅 Sala: Active Directory Basics

📚 O que aprendi na sala

Nesta sala, entendi os conceitos fundamentais do Active Directory (AD), que é amplamente utilizado em ambientes corporativos para gerenciamento de usuários, computadores e permissões.

Conceitos principais:
O que é o Active Directory
Estrutura básica de um domínio Windows
Função do Domain Controller (DC)
Diferença entre usuários, grupos e máquinas dentro do domínio
Autenticação em ambientes corporativos
Importância do AD em redes empresariais
🏢 Estrutura do Active Directory

Aprendi que o Active Directory funciona como uma central de gerenciamento de identidade, onde:

Usuários são organizados em domínios
Permissões são controladas por grupos
O Domain Controller valida autenticações
Políticas podem ser aplicadas em toda a rede (GPOs)

🔐 Segurança e impacto

O Active Directory é um dos principais alvos em ataques corporativos, pois:

Um único domínio comprometido pode dar acesso a toda a rede
Má configuração de permissões pode levar a escalonamento de privilégios
Ataques frequentemente exploram autenticação e credenciais fracas

🧠 Dificuldade e percepção

Dificuldade: Fácil / introdutória
Tempo de conclusão: variável
Foco: entendimento conceitual

🧩 Conclusão

Essa sala ajudou a entender como ambientes corporativos são estruturados e gerenciados.

O Active Directory é essencial tanto para:

Administração de redes Windows
Segurança ofensiva (Red Team / Pentest)

Mesmo sendo introdutório, o conteúdo é base para ataques mais avançados como:

Lateral movement
Privilege escalation
Credential dumping

⚙️ Observação pessoal

Apesar de simples, essa base é importante porque praticamente todo ambiente corporativo real utiliza Active Directory.
